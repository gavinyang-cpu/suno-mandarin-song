# 示例：创作一首中文抒情歌曲

## 歌曲主题：思念

这是一首关于思念的抒情歌曲，讲述在异地奋斗的年轻人对故乡和亲人的思念。

---

## 风格 Prompt

```
中文抒情, 75 BPM, 深情, 钢琴+弦乐, 温柔男声, 感人
```

---

## 生成指令说明

在 Suno AI 中使用：

1. **风格输入**：填入上面的风格 Prompt
2. **歌词输入**：填入歌词部分（去掉括号内的说明文字）
3. **歌词标签**：确保结构标签（如 `[主歌]`、`[副歌]`）在歌词中正确位置
4. **生成**：选择 V5 模型生成

---

## 使用本 Skill 的工作流

1. **确定主题**：爱情/励志/思乡/怀旧/友情
2. **选择风格**：参考 style-library.md
3. **创作歌词**：遵循 lyric-structure.md 的字数和押韵规则
4. **构建 Prompt**：组合风格 + 歌词
5. **生成测试**：生成 2-4 个版本
6. **筛选优化**：选择最佳版本
