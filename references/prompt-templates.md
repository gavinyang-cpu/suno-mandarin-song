## Remix 提示词

改变现有歌曲的风格：

```
原曲风格 → 新风格
例如：抒情 → 电子国风
```

**Prompt：**
```
电子国风, 128 BPM, 时尚, 古筝+合成器, 电子节拍, 潮流
```
