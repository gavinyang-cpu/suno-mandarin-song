# 中文歌词结构指南

## 中文歌词特点

中文歌词与英文不同，需要注意：
- **字数控制**：每句 6-12 个汉字为最佳
- **声调配合**：注意声调起伏与旋律配合
- **押韵规则**：按汉语拼音韵母押韵

## 歌曲结构

### 完整结构

```
[前奏] → [主歌1] → [副歌1] → [主歌2] → [副歌2] → [桥段] → [副歌3] → [尾奏]
```

### 常见结构（3-4分钟）

1. **主歌 → 副歌 → 主歌 → 副歌**（最简单的 ABAB 型）
2. **主歌 → 副歌 → 桥段 → 副歌**（ABABC 型）
3. **前奏 → 主歌 → 副歌 → 主歌 → 副歌 → 尾奏**

## 段落字数要求

### 主歌（Verse）

- 4-8 句，每句 7-9 字
- 叙事性强，描述故事背景
- 情感铺垫，为副歌做准备

**示例：**
```
窗外的雨滴落在心头（8字）
想起我们相识的那个秋（9字）
你的笑容像阳光一样（8字）
温暖了我孤单的守候（8字）
```

### 副歌（Chorus）

- 4-8 句，每句 6-8 字
- 情感高潮，记忆点强
- 通常是歌曲的核心句子

**示例：**
```
爱像星光闪烁（6字）
照亮黑暗的角落（7字）
你是我最深的执着（8字）
从未想过会错过（7字）
```

### 桥段（Bridge）

- 2-4 句，每句 7-9 字
- 情绪转折，新的视角
- 通常使用不同的旋律

### 前奏/间奏/尾奏

- 纯乐器演奏
- 可加入环境音或音效

## 中文押韵技巧

### 常见韵脚

| 韵母 | 例字 |
|------|------|
| ang | 光、忙、浪、望、伤 |
| ing | 情、星、听、醒、铭 |
| ong | 同、空、终、风、虹 |
| iao | 笑、跳、飘、遥、皎 |
| ie | 谢、别、帖、蝶、劫 |
| u | 书、图、苦、铺、珠 |
| ü | 雨、去、聚、遇、绪 |

### 押韵位置

- **偶数句押韵**：最常见，第二、四、六句押韵
- **每句押韵**：较少见，歌词较密
- **双句押韵**：两句为一组，每组押韵

## 情感曲线

### 主歌 → 副歌 能量递增

```
主歌1: [能量: 低] 叙述故事
主歌2: [能量: 中] 情绪发展
副歌1: [能量: 高] 情感高潮
桥段:   [能量: 中] 情绪转折
副歌2:  [能量: 最高] 最终高潮
```

### 歌词能量标记

在歌词中使用标签标注能量变化：
```
[能量: 低] - 叙述、回忆
[能量: 中] - 描写、抒情
[能量: 高] - 表达、呐喊
[能量: 上升] - 渐进高潮
```

## 结构标签使用

在创作时，使用结构标签标注段落：

```markdown
[主歌]
[情感: 忧伤]
回忆像一部老电影
播放着我们的从前
那些画面依然清晰
只是主角已经走远

[副歌]
[情感: 热烈]
[能量: 高]
你曾是我的全部
如今只能在梦里哭
如果时间能够倒流
我愿把爱重新守护
```

## 歌词创作注意事项

### ✅ 应该做

- 控制每句字数在 6-12 字之间
- 使用常见字词，避免生僻字
- 押韵要自然，不要强行凑韵
- 情感表达要真挚
- 句子要有画面感

### ❌ 不应该做

- 每句字数超过 15 字（容易吞字）
- 使用过于晦涩的诗词语言
- 强行押韵导致意思不通
- 句子太长没有节奏感
- 缺少情感起伏
