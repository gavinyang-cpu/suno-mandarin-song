# 中文字符标签参考

在 Suno AI 中使用中文标签来控制歌曲结构、情感和制作。

## 结构标签

| 标签 | 英文对应 | 用途 |
|------|----------|------|
| `[前奏]` | `[Intro]` | 乐器开场 |
| `[主歌]` | `[Verse]` | 叙事部分 |
| `[主歌1]` | `[Verse 1]` | 第一段主歌 |
| `[主歌2]` | `[Verse 2]` | 第二段主歌 |
| `[副歌]` | `[Chorus]` | 高潮 hook |
| `[副歌1]` | `[Chorus 1]` | 第一段副歌 |
| `[副歌2]` | `[Chorus 2]` | 第二段副歌 |
| `[桥段]` | `[Bridge]` | 情绪转折 |
| `[间奏]` | `[Interlude]` | 乐器过门 |
| `[尾奏]` | `[Outro]` | 结束部分 |
| `[结尾]` | `[End]` | 明确结束 |
| `[渐出]` | `[Fade Out]` | 渐弱结束 |

## 歌词格式示例

```markdown
[主歌]
[情感: 忧伤]
[能量: 低]
窗外的雨滴落屋檐（8字）
回忆像潮水涌上心间（9字）
那些年的温柔画面（8字）
如今只剩模糊的光点（9字）

[副歌]
[情感: 难过]
[能量: 高]
[声线: 沙哑]
你不在的每一天（7字）
世界失去了颜色（7字）
等待变成一种煎烤（8字）
泪水模糊了视线（7字）

[桥段]
[情感: 平静]
[能量: 中]
也许某天会相见（7字）
在某个城市的街角（9字）
轻声说一句 好久不见（9字）
```

## 使用技巧

1. **标签放前面**：重要标签放在歌词开头 3-5 行
2. **标签要简洁**：每个标签单独一行
3. **中英文混用**：可用英文标签，效果相同
4. **情感要一致**：整首歌的情感标签要协调
5. **能量渐变**：主歌→副歌能量递增
