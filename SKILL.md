## 中文歌曲创作工作流

### 1. 确定歌曲类型

| 类型 | 特点 | 适用场景 |
|------|------|----------|
| 中文流行 | 旋律流畅、情感丰富 | 流行榜单、抖音神曲 |
| 古风 | 诗意歌词、民族乐器 | 国风音乐、游戏配乐 |
| 民谣 | 叙事性强、吉他为主 | 校园民谣、情感故事 |
| 摇滚 | 力量感强、节奏激昂 | 乐队演出、热血时刻 |
| 抒情 | 旋律优美、情感深沉 | 影视主题曲、情歌 |

### 2. 中文歌词要点

**字数控制：**
- 每句 6-12 个汉字为最佳（保证演唱时值对齐）
- 避免句子过长导致 AI 演唱时吞字

**押韵规则：**
- 汉语拼音韵母相同即为押韵
- 常见韵脚：ang, ing, ong, iao, ie, u, ü

**情感表达：**
- 使用中文特有意象：月亮、故乡、江南、长城等
- 运用古典诗词手法：比兴、对仗、夸张

### 3. Prompt 构建公式

**风格 Prompt：**
```
[风格], [BPM], [情绪], [主要乐器], [人声类型], [制作特点]
```

**歌词格式：**
```
[结构标签]
[情感/能量指示]
中文歌词内容（每句6-12字）
```

### 4. 创作流程

1. **选题与立意** - 确定主题（爱情、励志、思乡、怀旧等）
2. **撰写歌词** - 按结构创作，保证字数和押韵
3. **构建 Prompt** - 组合风格标签 + 歌词
4. **生成测试** - 生成 2-4 个版本
5. **筛选优化** - 选择最佳版本，可使用 Extend 扩展
6. **后期处理** - Remaster 提升质量

## 常见问题

### Q: 中文歌词创作应该注意什么？

A: 
- 控制每句字数在 6-12 字之间
- 注意押韵，使歌曲朗朗上口
- 使用中文特有文化意象
- 避免过于复杂的诗意表达

### Q: 如何选择合适的 BPM？

A:
- 抒情慢歌：60-80 BPM
- 流行中速：100-120 BPM
- 动感快歌：130-150 BPM

### Q: 如何避免 "念经感"？

A:
- 歌词要有明确的情感起伏
- 句子要有长短变化
- 使用情感标签标注不同段落
