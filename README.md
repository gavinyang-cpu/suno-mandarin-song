# Suno Mandarin Song

使用 Suno AI 创作中文歌曲的专用 Skill。

## 概述

这个 Skill 为使用 Suno AI 创作中文歌曲（普通话/华语歌曲）提供了完整的工作流和参考资料。

## 功能

- 🎵 中文歌词结构指南
- 🎼 中文歌曲风格库
- 🏷️ 中文字符标签参考
- 📝 提示词模板

## 快速开始

1. 安装此 Skill
2. 阅读 SKILL.md 了解工作流
3. 参考 references/ 下的文件获取详细指导

## 目录结构

```
suno-mandarin-song/
├── SKILL.md                    # 主 Skill 文件
└── references/
    ├── lyric-structure.md      # 中文歌词结构
    ├── style-library.md       # 风格库
    ├── metatags.md            # 字符标签
    └── prompt-templates.md    # 提示词模板
```
